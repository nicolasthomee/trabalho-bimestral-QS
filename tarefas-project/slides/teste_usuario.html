Teste de Usuário: Foco na Experiência do Desenvolvedor

Definição
Avaliação da API por um "usuário" (o desenvolvedor cliente) que integrará a funcionalidade no seu aplicativo.
Cenário Testado
Criação de Tarefa com dados válidos e tratamento de erros de validação.
Foco na Usabilidade
Clareza das respostas HTTP (201, 422) e utilidade das mensagens de erro para o cliente integrador.
Questões Avaliadas
A API é intuitiva e fácil de usar?
As mensagens de erro são claras e úteis?
O fluxo de autenticação é bem documentado?
