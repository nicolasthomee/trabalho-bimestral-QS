Visão Geral da API de Tarefas

Recurso Principal
Tarefa (Model, Migration, Controller)
Funcionalidades
Listagem (GET /api/tarefas) e Criação (POST /api/tarefas)
Autenticação
Implementada com Laravel Sanctum (Tokens de API)
Rotas de Login
POST /api/login para geração do token
Proteção de Rotas
Middleware auth:sanctum para proteger o recurso tarefas
Banco de Dados
SQLite para desenvolvimento e testes
